# SEHS4507_Research_Method
# 🧭 一、研究主題與方向設定  
（參考 Lecture 2A：Steps in Planning Research Projects）

## 1️⃣ 選題來源
- 個人興趣與社會議題結合：NDE（瀕死經驗）是一個橫跨心理學、宗教研究及醫療社會學的熱門主題。  
- 知識缺口（Knowledge Gap）：過去研究多著重於 NDE 的醫學特徵（如幻覺、生理變化），但較少討論其後續對人格變化、價值觀、宗教信仰及死亡恐懼的影響。

## 2️⃣ 研究問題
- 研究問題：NDE 是否會改變個體的人格特質、價值觀、宗教信仰或死亡恐懼？  
- 假設（參考 Lecture 2, slide “Research question vs Hypothesis”）：
  - H₁：經歷過 NDE 的人具有與一般人顯著不同的人格特質與宗教信仰傾向。  
  - H₂：NDE 可減低個體對死亡的恐懼程度。

# 🧩 二、文獻回顧（Lecture 4A: Reviewing the Literature）

## 步驟
1. 搜集已發表的學術文章（Google Scholar, APA database）  
2. 關鍵字：Near-Death Experience, Personality Change, Death Anxiety, Religious Belief Shift, Value Transformation  
3. 比較不同理論觀點：
   - 心理學觀點（e.g. Moody, Greyson）  
   - 臨床醫學觀點（如生理因素造成 NDE 幻覺）  
   - 宗教與靈性觀點（如視 NDE 為靈性覺醒經驗）  
4. 評析文獻：
   - 找出先前研究的相似與矛盾點  
   - 指出研究空白（例如：缺乏量化分析 / 缺乏亞洲樣本）

# 💡 三、研究目的與重要性  
（Lecture 2 & Lecture 1）

## 研究目的
- 了解 NDE 對人格與信仰的潛在心理效應。  
- 探索是否存在「減少死亡焦慮」的明顯趨勢。  
- 對臨床心理或靈性輔導提供參考依據。

## 研究貢獻（Significance）
- 理論層面：補充 NDE 對心理適應影響的研究缺口。  
- 實務層面：幫助臨床心理師了解病患在「瀕死後存活」的創傷後成長現象。

# ⚙️ 四、研究設計與方法  
（Lecture 3, 5, 6, 7, 8）

## 1️⃣ 研究設計（Research Design）
- 採取混合式研究（Mixed Methods）（Lecture 3）：  
  - 質性：半結構式訪談 → 深入了解 NDE 體驗後的心理轉變。  
  - 量化：自填式問卷 → 測量人格（Big Five Inventory）、宗教信仰、死亡焦慮（e.g. Death Anxiety Scale）。

## 2️⃣ 研究對象與抽樣（Sampling）（Lecture 7）
- 目標族群：18 歲以上、有親身經歷 NDE / 醫療搶救生還者。  
- 抽樣方法：便利抽樣（Convenience Sampling）或滾雪球抽樣（Snowball Sampling）（若樣本難尋）。  
- 目標樣本：預期 20–30 位（質性深度為主，量化結果作探索性分析）。

## 3️⃣ 數據收集方法（Data Collection）（Lecture 6）
- 質性：半結構式訪談，每人 30–45 分鐘，問題例：
  - 「請描述你的瀕死經驗？」  
  - 「你覺得經歷後對人生、死亡的態度有什麼改變？」  
- 量化：Likert scale 問卷（1＝非常不同意 到 5＝非常同意）。  
  - 例題：「我不再害怕死亡。」／「我比以前更相信某種更高的力量。」

## 4️⃣ 分析方法（Lecture 8 & 9）
- 質性資料：主題分析（Thematic Analysis）。  
- 量化資料：描述統計（平均、標準差）、交叉分析（crosstab）及假設檢定（T-test, Chi-square）。  
- 工具：SPSS 或 Excel（或 R 為替代）。

# 🔐 五、倫理考量（Lecture 2B）
- 取得書面同意（Consent Form），清楚說明：  
  - 參與自願，可隨時退出；  
  - 資料僅用於學術研究；  
  - 保密處理，匿名化訪談資料。  
- 涉及生死議題，需尊重受訪者情緒與文化信仰背景，訪談過程應準備情緒支持或轉介機制。  
- 所有記錄與問卷數據保存於加密檔案中，限研究成員使用。

# 🧮 六、資料整理與呈現（Lecture 9）
研究報告應包括：

- 人口統計資料：表格與統計描述（性別、年齡、是否有宗教背景）。  
- 問卷分析結果：長條圖、交叉表（如人格變化 vs 宗教信仰）。  
- 訪談摘錄：以引號呈現代表性語句，進行主題分析。  
- 整合分析：對照質性與量化結果，提出趨勢與例外情況。

（示例表格欄位：類別 / 使用方法與呈現方式）
- 人口統計資料 — 表格與統計描述  
- 問卷分析結果 — 圖表（長條圖）、交叉表、統計檢定結果  
- 訪談摘錄 — 代表性引言 + 主題說明  
- 整合分析 — 質性-量化對照、綜合討論

# 🔚 七、結論與建議（Lecture 9）

## 結論（範例）
- 總結：評估 NDE 是否在心理層面造成持久影響（如人格、信仰與死亡恐懼的變化）。

## 建議
- 未來可擴大樣本到不同文化背景（跨文化比較）。  
- 增加長期追蹤（longitudinal study）以檢視效果持久性。

## 侷限性
- 樣本小且非隨機，限制外推性。  
- 資料依賴自我回憶可能有記憶偏誤。

- Moody, R. (1975). Life After Life. Mockingbird Books.
